基于物理的着色（PBS）技术，
->
就是指按照物体表面各种材质与光线的相互作用的物理学原理，用数学建模的方法建立起一系列光照模型，
从而使得渲染出来的图像更接近真实世界中的外观，更有质感的一种技术。
-->>
实时渲染领域中的几种光照模型（简化的方式近似地模拟）。、


【 ****** 漫反射和Lambert光照模型 ****** 】
 Lambert光照模型定义为漫反射光、环境光、自发光之和。


【 ****** 镜面反射和Phong光照模型 ****** 】
 Phong光照模型可视为Lambert光照模型加上镜面反射项。

存在问题：
当观察点越靠近入射光方向时，就会出现反射向量和视线向量的夹角大于 90°，
从而出现两向量的点积为负数的情况。这就是为什么计算公式中需要在向量点积值和 0 之间做一个 max 判断操作。
这种硬性地控制取值就导致，远处黑暗处的过渡显得比较突兀，突然间而不是逐渐地变黑下来。


【 ****** Blinn-Phong光照模型 ****** 】
为解决Phong存在的问题，
->
提出半角向量：入射光线反向向量L与观察向量V的夹角中分线向量。
-->>
Blinn-Phong 光照模型中，不是用反射向量R与观察向量V进行点积，而是用表面法向量N和半角向量H进行点积。


【 ****** 基于物理的光照模型的相关概念 ****** 】

// **** 光的折射和折射率：
->
某介质的某处对光的折射率（refractive index）即光在真空中的传播速度与光在此介质某处的传播速度之比。
介质某处的折射率越高，使入射光发生折射的能力越强。
如果某种介质各向同性，它处处的折射率都相等，则这种介质就称为均匀介质（homogeneous medium）。
介质的折射率与介质自身的电磁性质密切相关。
另外，同一种介质对不同频率的光的折射率也不一样，这种现象称为色散（chromatic dispersion）。

// **** 散射：
->
光线在均匀介质中总是沿着直线传播，但是在非均匀介质（heterogeneous medium）中，
因为同一种介质中的不同部分对光的折射率不同，所以光线将不严格地沿着直线传播。
如果介质的折射率缓慢且连续变化，光线传播的路径就会逐渐变成曲线；
如果折射率突然改变，如光线碰到介质中的其他杂质，光线就会发生散射（scattering），
即一束光会被分成很多束，向不同方向传播。
如果不考虑在传播过程中光被吸收的能量，散射光的总能量保持不变。

// **** 双向反射分布函数（bidirectional reflectance distribution function，BRDF）：
->
BRDF描述的就是表面入射的光线的能量和从表面出射的光线的能量的比例关系。
用数学语言可描述为：当沿立体角ωi的方向入射到点p的光辐射亮度为Li(p,wi)时，
计算朝向观察者的立体角ωo方向上有多少辐射亮度Lo(P,wo)离开点p。
-->>
同一表面对不同频率的光（不同颜色的光）的 BRDF 反射率可能不一样，因此 BRDF 和光的频率有关。
在图形学中，如果使用 RGB 表示光的颜色，通常 3 个分量各有自己的 BRDF 函数。
--->>>
基于物理的标准光照模型：Cook-Torrance模型

// **** 菲涅尔反射：
一种近强折射远强反射的效果称为菲涅尔反射。
->
金属有些特有的性质，引出金属工作流，金属度。

// **** 微表面和Cook-Torrance模型
->
法线分布函数：
当给定一个半角向量h时，法线分布函数会返回其法线「恰好是」是h的微表面的个数占待光照计算表面区域中微表面总数的比例。
->
几何衰减因子函数：
该函数输入值是入射光线方向ωi和光线的出射方向ωo，输出值为光线未被遮挡或蒙盖，能从ωi出射到ωo方向的比例。
在编程实践中，光线的出射方向ωo通常指片元到当前摄像机的连线向量。

// **** 定义菲涅尔方程：
->
完美镜面并不会将所有的光线反射掉，而是一部分被反射，一部分被折射，反射比遵循菲涅尔方程。
->
菲涅尔方程定义为法向量n（可用半角向量h代替）和入射光线方向向量wi的函数。（Unity3D 使用 Fresnel-Schlick函数近似模拟）

// **** Cook-Torrance BRDF模型：
->
Torrance-Sparrow 基于微表面理论，用上述的法线分布函数D(h)、几何衰减因子函数G(ωi, ωo)、菲涅尔方程函数F(h, ωi)
和待光照计算的表面的宏观法线N（用于计算半角向量h）建立了 BRDF 模型。
-->>
除去镜面反射部分的 BRDF 模型之外，在渲染时还应包含漫反射部分的 BRDF 模型。
--->>>
Unity 3D 并没有直接使用 Cook-Torrance 的漫反射 BRDF 模型，而是使用了 Disney 提出的模型。
（位于UnityStandardBRDF模型中DisneyDiffuse函数中）
